# 表格穿梭框组件


<htTableTransfer
  :width="'100%'"
  :height="'80vh'"
  :leftDataList="leftDataList"
  :targetList="rightDataList"
  :tableHeight="'59vh'"
  :columnData="columnData"
  @setTargetList="setTargetList"
  >
</htTableTransfer>

....data

  leftDataList: [//...
  {
    id: 40,
    date: '2016-05-03',
    name: '王小虎',
    address: '上海市普陀区金沙江路 1516 弄'
  }
  ],
  columnData: [
    {
      prop: 'name',
      label: '名称',
      width: '130',
      sortable: true
    },
    {
      prop: 'date',
      label: '日期',
      width: '210',
      sortable: true
    },
    {
      prop: 'address',
      label: '地址',
      width: '210',
      sortable: true,
      tooltip: true
    }
  ],
  rightDataList: [],


...methods

setTargetList (list) {
  console.log('list', list)
}

# 属性(Attributes)

参数 说明 类型 默认值 可选值
width 宽度 String '800px' -
height 高度 String '500px' -
tableHeight 表格高度 String '500px' -
titles 标题 Array ['源数据', '目标源数据'] -
leftDataList 左侧数据源 Array [] -
targetList 右侧数据源 Array [] -
columnData 表格列 Array [] -

| buttonTexts| 穿梭按钮文案 | Array | ['移除', '关联'] | - |

# 方法(Methods)

方法名 说明 参数
setTargetList 穿梭事件处理 list

# 插槽 Slot

name 说明
left-query 插入至左侧顶部的查询表单
left-footer 插入至左侧底部的分页
right-query 插入至右侧顶部的查询表单
right-footer 插入至右侧底部的分页

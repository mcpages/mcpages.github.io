# 按钮组(内置和自定义按钮)

表格组件内置了三个按钮组，分别是：新增按钮、导入按钮、导出按钮、删除按钮

# 新增按钮使用方法

当add按钮是true时，需要在新增弹框中插入内容


<htTableQuery
  ref="htTableQuery"
  :tableData="tableData"
  :columnData="columnData"
  :pagination="pagination"
  @handleSizeChange="handleSizeChange"
  @handleCurrentChange="handleCurrentChange"
  :add="true"

>
  <!-- 新增弹框内容 start-->
  <template slot="addDialogContent">
      <htFormSubmit
      :formData="addForm"
      @submitForm="submitForm"
      >
      </htFormSubmit>
  </template>
  <!-- 新增弹框内容 end-->
</htTableQuery>

    beforeUpload (file) {
      const isExcel =
        file.name.split('.')[1] === 'xlsx' || file.name.split('.')[1] === 'xls'
      const isSize = file.size / 1024 / 1024 < 1
      if (!isExcel) {
        this.$message({
          message: '只能上传xls或xlsx文件!',
          type: 'error'
        })
      }
      if (!isSize) {
        this.$message({
          message: '上传文件大小不能超过 1MB!',
          type: 'error'
        })
      }
      return isExcel && isSize
    },

# 属性(Attributes)

参数 说明 类型 默认值 可选值
tableData 表格数据 Array [] -
columnData 表格列配置数据 Array [] -
pagination 分页配置 Object {currentPage: 1,pageSize: 100,pageSizes: [100, 200, 300, 400],total: 400} -
add 是否展示新增按钮 Boolean false -
lead 是否展示导入按钮 Boolean false -
derive 是否展示导出按钮 Boolean false -
del 是否展示删除按钮 Boolean false -
advanced 是否高级查询 Boolean false -
tableColumnSelected 展示选中的表格列 [] [] -

# 方法(Methods)

方法名 说明 参数
handleCurrentChange 页码改变改变 pagination对象
handleSizeChange 分页大小改变 pagination对象
beforeUpload 上传文件前的操作 file
uploadChange 上传文件改变 file
uploadFile 上传方法覆盖 file
exportSelected 导出按钮方法 rows
delSelected 删除按钮方法 rows
tableColumnSelectedChange 高级设置修改了展示表格列 values

# 插槽属性 Slot

name 说明
btns 自定义按钮组
advance 插入至右侧高级设置，可以根据选中的checkbox显示隐藏哪些属性，详细使用方法可以参照拖拽排序按钮组
expand 行扩展
operate 操作列扩展
addDialogContent 新增按钮弹框

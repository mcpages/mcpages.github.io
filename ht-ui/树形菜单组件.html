# 属性(Attributes)

参数 说明 类型 默认值 可选值
defaultProps 属性映射 Object {children: 'children',label: 'name'} -
nodeKey 唯一标识key String 'menuId' -
showCheckbox 是否显示checkbox Boolean false -
treeHeight 高度 Number height-500 -
treeWidth 宽度 Number 237 -
expandedKeys 默认展开的树节点，ids Array [] -
draggable 是否可拖拽 Boolean false -
showFilter 是否过滤 Boolean false -
showRightOptions 是否显示右侧图标 Boolean false -
showRightMenu 是否可以右击 Boolean false -
contextMenu 右击菜单 Array false -
rightFlag 右击菜单选中的值 Array false -

# 方法(Methods)

方法名 说明 参数
handleNodeClick 节点单击左键 data
handleRightClick 右键单击 event, obj, node, self
handleClickMenu 选中菜单 item
handleNodeExpand 点击扩展 data, node, self
handleClickMenu 选中菜单 item
setCurrTreeNode 设置当前节点 item
handleDragStart 同element ui -
handleDragEnter 同element ui -
handleDragLeave 同element ui -
handleDragOver 同element ui -
handleDragEnd 同element ui -
handleDrop 同element ui -
filterNode 设置当前节点 item
append 新增节点 data
edit 编辑节点 { node, data }
remove 删除节点 { node, data }
handleCheckChange 点击checkbox变化 { data, checked, indeterminate }

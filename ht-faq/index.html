# 权限控制与路由规则

1、系统设置中资源管理，创建资源的URL就是页面存放路径

2、在views下面创建相匹配的文件夹和文件名

3、路由中传参，不重要的参数推荐放在url中，涉及id，非加密的code，放在pinia或者params中

  // 字符串
  router.push('home')
  
  // 对象
  router.push({ path: 'home' })
  
  // 命名的路由
  router.push({ name: 'user', params: { userId: '123' }})
  
  // 带查询参数，变成 /register?userId=123
  router.push({ path: 'register', query: { userId: '123' }})
